Quick laundry guide (for the rest of us)

A starter cheat sheet — Mum probably knows this already, but it's here for everyone else.

LoadTempNotes
Whites40°C / warmWhite socks, undies, t-shirts. Add a stain-remover for the stubborn stuff.
Lights30°CPale colours — beige, pastels, light grey.
Darks30°C / coldBlack, navy, dark grey. Inside out keeps the colour.
Colours30°CAnything bright and mixed.
Reds30°CFirst few washes — keep separate, they bleed.
DelicatesCold, gentleBras, silk, anything with "hand wash" on the tag. Use a mesh bag.
Towels60°CHot enough to kill bacteria. No fabric softener — it stops them absorbing.
Bedding60°CStrip beds the same morning so it all goes in together.
Sportswear30°CCold protects the elastic. Skip softener.

Golden rules: separate by colour, zip up zips, empty pockets, don't overfill the drum (you should be able to fit a hand on top).
